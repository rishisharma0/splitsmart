# SplitSmart

A group expense tracker that helps friends split bills and settle up with the minimum number of transactions. Built as a self-learning project after a frustrating trip-expense settlement with roommates.

## Tech Stack

- **Frontend:** React (Vite), Tailwind CSS, Axios
- **Backend:** Node.js, Express, MongoDB (Mongoose)
- **Auth:** JWT + bcrypt
- **Deployment:** Vercel (frontend), Render (backend), MongoDB Atlas (database)

## Features (planned)

- [ ] User signup / login with JWT authentication
- [ ] Create groups and add members
- [ ] Add expenses with equal or unequal splits
- [ ] View group balances — who owes whom
- [ ] Debt simplification — minimum transactions to settle up

## Status

🚧 In progress — building from scratch over 30 days while learning the MERN stack.
